# To-Do List Web Application

A **web-based To-Do List application** built with **Python** and **Django/Flask/FastAPI**, providing full **CRUD functionality** (Create, Read, Update, Delete) for task management. This app allows users to efficiently manage tasks through a clean and intuitive web interface and RESTful APIs.

---

## Features

- Create, view, update, and delete tasks
- User-friendly web interface
- RESTful API endpoints for integration
- Stores tasks in **SQLite** database
- Responsive and minimal design

---

## Tech Stack

- **Backend:** Python, Django / Flask / FastAPI  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite
